# ParseBox.rPi

ParseBox is a root-level npm scaffold for two aligned goals:

1. A VitePress docs project that can build for GitHub Pages
2. A minimal Raspberry Pi kiosk-facing site output

## Usage

```bash
npm install
npm run docs:dev
npm run docs:build
```

Built output is generated by VitePress and includes a kiosk page at `/kiosk/`.
