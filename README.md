# ParseBox.rPi

ParseBox is a root-level npm scaffold for two aligned goals:

1. A VitePress docs project that can build for GitHub Pages
2. A separate root-level minimal Raspberry Pi kiosk-facing page

## Usage

```bash
npm install
npm run docs:dev
npm run docs:build
```

Docs build output is generated by VitePress.

The kiosk page is intentionally separate from docs at:

- `/kiosk/index.html`
